# Changelog - FinanceHub

Todas as mudanças importantes deste projeto serão documentadas neste arquivo.

---

## [v0.0.19] - 2025-01-15

### ✨ Novas Funcionalidades
- **Colorização Automática de Categorias**: Nova ferramenta no painel administrativo para aplicar cores padrão nas categorias do sistema
- **Reorganização do Painel Admin**: Melhor organização das ferramentas administrativas com abas dedicadas

### 🔧 Melhorias
- **Relatórios Financeiros**: Corrigidos problemas nos gráficos "Receitas vs Despesas" e "Fluxo de Caixa" que não exibiam dados do mês atual
- **Interface Admin**: Texto do menu lateral alterado de "Personalizar SaaS" para "Personalizar LOGO" para maior clareza
- **Uploads de Logo**: Melhorada a configuração de permissões para upload de logos em ambiente de produção
- **Credenciais de Sistema**: Adicionado suporte para configuração personalizada de usuário administrador via variáveis de ambiente

### 🐛 Correções
- **DOM Validation**: Corrigidos warnings de estrutura HTML nos componentes de relatórios
- **Compatibilidade de Temas**: Melhorada a compatibilidade entre nomes de meses em português nos gráficos
- **Permissões de Arquivo**: Correção automática de permissões das pastas de upload durante inicialização do sistema

### 🎨 Interface
- **Nova Seção Admin**: Adicionada seção dedicada "Colorizar Categorias" no menu administrativo
- **Layout Aprimorado**: Melhor organização visual das ferramentas administrativas
- **Feedback Visual**: Melhorados os indicadores de carregamento e mensagens de status

### 📋 Organização
- **Estrutura de Menus**: Reorganização das opções administrativas em categorias mais intuitivas
- **Fluxo de Trabalho**: Simplificação do acesso às ferramentas de personalização do sistema
